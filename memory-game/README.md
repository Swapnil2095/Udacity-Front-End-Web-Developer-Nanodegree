## Instructions

Click the cards to find a match. Match all 16 cards as fast and best as you can! <br/>
It's a \#SPA \#SinglePageApp

* HTML5
* CSS3
* Bootstrap
* Font-Awesome


#### How To Run

* clone this repo or download and extract via zip
* open the folder
* right click `index.html` and choose a browser
* play the game!
